# OpenLayers map integration for Inkdrop

Supports OpenStreetMaps, Bing Maps and Google Maps.

## Install
```
ipm install ol
```

## Example
    ```ol
    source: bing

    view:
      # The center coordinates can be taken from Google Maps
      center: [34.652500, 135.506302]
      zoom: 9
    ```
